Some titles for possible review

I don't intend this to be exhaustive and I certainly don't expect us to cover more than a fraction of what is listed here. These are just books that might lead to worthwhile discussions. Most of them are fairly recent. Others have been around for decades and are probably familiar titles to people in this group. Others have not yet been published.

Clojure Programming Emerick, Carper, Grand
Programming Clojure Halloway
Programming in Scala Odersky, Spoon, Venners
Scala in Depth Suereth
Scala for the Impatient Horstmann
Programming Scala Wampler, Payne
The Well-Grounded Java Developer Evans, Verburg
Jave Generics And Collections Wadler, Naftalin
Real World Haskell O'Sullivan, Goerzen, Stewart
Learn You A Haskell For Great Good Lipovača
ANSI Common Lisp (and On Lisp) Graham
Practical Common Lisp Seibel
